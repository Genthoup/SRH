PYTHON-DJANGO-REST-ANGULAR JS
===

An example to use  python-django-rest-angular on a project

REQUIREMENTS
===

* Python 2.6.x
* Django 1.5
* Angular JS 1.1.5
* Restangular 1.07
* Jquery 1.9.1
* Virtualenv
* South
* django-allauth
* django-filter
* django-grappelli
* djangorestframework
* oauth2
* oauthlib
* openpyxl
